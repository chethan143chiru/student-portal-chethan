# 🚀 Smart Student Service Hub – AI-Ready Portal

> A full-stack web application designed to simulate a real-world student service system with dynamic data handling, clean UI, and structured backend architecture.

---

## 📌 Project Overview

The **Smart Student Service Hub** is a modern web-based platform that allows students to request services, submit queries, and interact with a centralized system.

It integrates **frontend + backend + JSON database** to demonstrate a complete development workflow.

This project is built with a focus on:

* Simplicity
* Real-world functionality
* Clean architecture
* Scalable design approach

---

## 🎯 Objectives

* Develop a multi-page responsive web application
* Implement frontend-backend communication
* Use JSON file as a database
* Handle API requests using GET and POST methods
* Demonstrate real-time data storage and retrieval
* Follow proper Git workflow and project structure

---

## 🧠 System Architecture

```
User Interface (Frontend)
        ↓
JavaScript (Fetch API)
        ↓
Node.js + Express (Backend)
        ↓
JSON File (Database)
        ↓
Admin Dashboard (Data Display)
```

---

## 🧩 Features

✨ Dashboard-style Home Page
✨ Multi-page navigation (6 pages)
✨ Service request form (dynamic submission)
✨ Admin panel for viewing submitted data
✨ JSON-based persistent storage
✨ REST API integration (GET & POST)
✨ Basic login authentication
✨ Clean and responsive UI

| Versioning | Git & GitHub          |

## 🔄 Application Workflow

### 📩 Data Submission

* User fills the service request form
* JavaScript sends data using POST request
* Backend receives and stores data in JSON

### 📊 Data Retrieval

* Admin page sends GET request
* Backend reads JSON file
* Data is displayed dynamically
